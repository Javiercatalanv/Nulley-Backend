# Nulley-Backend
Nulley is a web and mobile solution designed to centralize and visualize scientific publication data from university researchers. Currently, publication records are scattered across multiple external platforms and manually consolidated in spreadsheets, making it difficult to track progress, analyze trends, and support decision-making processes.

This project aims to address these challenges by building a unified system that aggregates data from various sources and stores it in a structured database. The platform will provide interactive dashboards that display key metrics such as the number of publications over time, enabling users to compare performance across different periods and categories. Additionally, it will include researcher profile pages that consolidate relevant information and provide direct links to their external academic and professional networks.

The system is designed for academic administrators, researchers, and external users interested in evaluating research output and institutional progress. It prioritizes usability, performance, and scalability, ensuring that the platform can efficiently handle growing datasets and an increasing number of contributors over time.
Ultimately, this platform seeks to improve visibility into scientific productivity, facilitate data-driven decision-making, and streamline the management of academic research information.
